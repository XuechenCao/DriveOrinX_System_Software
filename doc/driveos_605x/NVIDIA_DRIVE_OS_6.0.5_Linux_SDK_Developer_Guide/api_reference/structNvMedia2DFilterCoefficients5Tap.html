NvMedia API for DRIVE OS 6.0 » 2D Processing

Detailed Description

Coefficients values structure for 5-tap custom filter.

This type holds the list of coefficients values for a 1D polyphasic filter with 5 taps and 32 phases.

This struct itself can be considered a POD type, so it does not have any functions to create/destroy it.
